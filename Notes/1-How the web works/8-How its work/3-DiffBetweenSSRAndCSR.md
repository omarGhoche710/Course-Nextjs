تخيّل الموقع مثل **مطعم** 🍽️.

---

# 1️⃣ Client Rendering

هذا الأسلوب الذي يستخدمه غالبًا React.

### ماذا يحدث؟

1. المتصفح يطلب الصفحة.
2. السيرفر يرسل **ملف JavaScript كبير**.
3. المتصفح يشغّل JavaScript.
4. JavaScript يبني الصفحة داخل المتصفح.

يعني **المتصفح هو الذي يبني الصفحة**.

### الشكل:

```
User -> Request
Server -> JavaScript
Browser -> JavaScript builds the page
```

### مثال

تفتح الموقع:

```
example.com
```

في البداية ترى صفحة **فارغة لحظة**
ثم يظهر المحتوى بعد أن يعمل JavaScript.

📌 هذا يسمى **Client Side Rendering (CSR)**
لأن **الـ Client (المتصفح)** هو الذي يرندر الصفحة.

---

# 2️⃣ Server Rendering

هنا العكس.

السيرفر هو الذي يبني الصفحة.

### ماذا يحدث؟

1. المستخدم يطلب الصفحة.
2. السيرفر يبني **HTML جاهز بالكامل**.
3. المتصفح يعرضه فورًا.

### الشكل

```
User -> Request
Server -> builds HTML
Browser -> shows page
```

يعني الصفحة **تصل جاهزة**.

📌 هذا يسمى **Server Side Rendering (SSR)**.

---

# 3️⃣ أين يأتي دور Next.js

Next.js يسمح لك باستخدام **الطريقتين معًا**.

يمكنه:

### SSR

السيرفر يبني الصفحة أولًا.

### ثم React يعمل على المتصفح

لإضافة التفاعل.

هذه العملية اسمها:

**Hydration**

---

# 4️⃣ مثال واقعي

تخيل صفحة منتج في متجر.

### CSR (React فقط)

1. الصفحة تفتح
2. فارغة قليلاً
3. JavaScript يجلب البيانات
4. يظهر المنتج

---

### SSR (Next.js)

1. الصفحة تفتح
2. المنتج ظاهر فورًا
3. بعد ذلك React يضيف التفاعل

---

# 5️⃣ لماذا يستخدم الناس Next.js ؟

لأنه يعطيك:

✅ سرعة تحميل أفضل
✅ SEO أفضل (جوجل يرى المحتوى)
✅ يمكن استخدام React بسهولة

## 1️⃣ المتصفح يرسل طلب (Request)

المتصفح يقول للسيرفر:

```
أعطني الصفحة
```

---

## 2️⃣ السيرفر يرسل الملفات

السيرفر يرسل للمتصفح عادةً:

```
HTML
CSS
JavaScript
```

### HTML

هو **هيكل الصفحة**.

مثال:

```html
<h1>Hello</h1>
<button>Click</button>
```

---

### CSS

هو **شكل الصفحة**.

مثال:

```css
h1 {
  color: red;
}
```

---

### JavaScript

هو **التفاعل**.

مثال:

```javascript
button.onclick = () => {
  alert("Hello");
};
```

---

## 3️⃣ المتصفح يبني الصفحة

المتصفح يأخذ هذه الملفات ويقوم:

1️⃣ يقرأ **HTML**
2️⃣ يطبّق **CSS**
3️⃣ يشغّل **JavaScript**

فتظهر الصفحة للمستخدم.

---

## أين الفرق الذي يهمنا؟

### في Client Rendering

السيرفر يرسل:

```
HTML فارغ تقريبًا
+ JavaScript كبير
```

و **JavaScript يبني الصفحة داخل المتصفح**.

---

### في Server Rendering

السيرفر يرسل:

```
HTML جاهز بالكامل
```

والمتصفح يعرضه مباشرة.

---

💡 مثال سريع:

#### Client Rendering (React)

```html
<div id="root"></div>
```

والـ JavaScript يبني كل الصفحة.

---

#### Server Rendering

```html
<h1>Product</h1>
<p>Price: 20$</p>
```

الصفحة **جاهزة مباشرة**.
