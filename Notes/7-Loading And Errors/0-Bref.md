# 🎭 The Art of Failing Gracefully

> ⚡ Official repository for the **TechTalks Ultimate Next.js 16 Course**

This repository teaches how modern applications **handle failure using the Next.js App Router**.

---

## 🧠 What You’ll Master

* Segment error boundaries (`error.tsx`)
* Global crash handling (`global-error.tsx`)
* Expected vs unexpected errors
* 404 handling (`not-found.tsx`, `global-not-found.tsx`)
* Designing resilient UI systems

---

## 🏗️ The Core Idea

> Next.js does NOT handle errors with logic…
>
> It handles them with **file-based architecture**.
