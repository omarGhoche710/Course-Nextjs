تمام 😄، خلينا نشرح هذا الكود خطوة خطوة لأنه **RootLayout** في Next.js 13/16 مع TypeScript، وبيجمع بين **Layout عام + خطوط + Navbar/Footer + الصفحة نفسها**.

---

# 1️⃣ الاستيراد (Imports)

```ts
import type { Metadata } from "next";
import { Geist, Geist_Mono } from "next/font/google";
import "./globals.css";
import Navbar from "@/components/Navbar";
import Footer from "@/components/Footer";
```

* `Metadata` → لتحديد **عنوان الموقع ووصفه**
* `Geist` و `Geist_Mono` → خطوط Google يتم تحميلها بطريقة متوافقة مع Next.js
* `globals.css` → ملف CSS عام للمشروع
* `Navbar` و `Footer` → Components لإعادة استخدام شريط التنقل والتذييل

---

# 2️⃣ إعداد الخطوط

```ts
const geistSans = Geist({
  variable: "--font-geist-sans",
  subsets: ["latin"],
});

const geistMono = Geist_Mono({
  variable: "--font-geist-mono",
  subsets: ["latin"],
});
```

* Next.js 13+ يدعم استيراد خطوط Google مباشرة باستخدام **next/font/google**
* هنا أنشأنا **متغير CSS** لكل خط لتطبيقه على الموقع
* `subsets: ["latin"]` → لتقليل حجم الخط والتحميل السريع

---

# 3️⃣ Metadata

```ts
export const metadata: Metadata = {
  title: "Introduction To Next.js 16",
  description: "Generated by create next app",
};
```

* هذا يستخدمه Next.js تلقائيًا لتوليد `<title>` و `<meta>` في `<head>`
* يمكنك إضافة وصف، كلمات مفتاحية، أيقونات، إلخ

---

# 4️⃣ تعريف RootLayout

```ts
export default function RootLayout({
  children,
}: Readonly<{ children: React.ReactNode }>) {
```

* `children` → كل محتوى الصفحة الذي سيظهر بين **Navbar** و **Footer**
* `Readonly` + `React.ReactNode` → TypeScript للتأكد من صحة النوع وعدم تعديل `children`

---

# 5️⃣ HTML الأساسي

```tsx
<html lang="en">
  <body className={`${geistSans.variable} ${geistMono.variable} antialiased`}>
    <Navbar />
    {children}
    <Footer />
  </body>
</html>
```

* `<html lang="en">` → لغة الصفحة
* `className` → تطبيق الخطوط المستوردة وميزة `antialiased` لتحسين وضوح النصوص
* **Navbar** و **Footer** → ثابتين لكل صفحة
* `{children}` → هنا تظهر كل صفحات أو Layoutات فرعية

---

# 6️⃣ النتيجة النهائية

كل صفحة ضمن `app/`:

* سترث **RootLayout**
* سيظهر **Navbar + Footer**
* الخطوط ستطبق تلقائيًا على كل النصوص
* Metadata سيتم إضافته تلقائيًا للـ HTML

---

💡 **الخلاصة:**

* هذا هو **الLayout الأساسي لكل الموقع**
* يربط بين **المظهر العام (CSS + Fonts)** و **الهياكل المشتركة (Navbar/Footer)**
* `{children}` يسمح لأي صفحة فرعية أو Layout إضافي أن يظهر داخل هذا الهيكل
