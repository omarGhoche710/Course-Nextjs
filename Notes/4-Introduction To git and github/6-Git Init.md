## 📝 Git Init — ملاحظات مهمة

### 🔹 شو هو `git init`؟

أمر من **Git** بيحوّل الفولدر تبعك إلى:

👉 **Repository (repo)**

---

## 🔹 شو بيعمل بالضبط؟

لما تكتب:

```bash
git init
```

✅ بيعمل:

* إنشاء فولدر مخفي اسمو `.git`

❌ ما بيعمل:

* فولدر اسمو init
* ولا بيغير ملفاتك

---

## 🔹 شو هو `.git`؟

* فولدر مخفي
* فيه كل معلومات المشروع:

  * commits
  * history
  * branches

💡 يعني هو قلب المشروع 🧠

---

## 🔹 قبل وبعد

### قبل:

```
my-project/
  index.html
```

### بعد:

```
my-project/
  index.html
  .git/
```

---

## 🔹 أهم فكرة 💡

👉 أول ما تعمل `git init`
صار عندك repo جاهز تستخدم فيه Git

بس لسا:

* ما حفظت شي
* ما في commits

---

## 🔹 متى تستخدمه؟

* أول مرة تبلّش مشروع جديد
* أو إذا عندك مشروع قديم وبدك تضيفله Git

---

## 🔥 الخلاصة السريعة

`git init` =
👉 “ابدأ تتبع هيدا المشروع باستخدام Git”
