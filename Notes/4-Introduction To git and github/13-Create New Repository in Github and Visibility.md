تمام 👌 هيدي نقطة مهمّة بـ GitHub 🔥

# 🧠 شو يعني **visibility** بـ GitHub؟

👉 يعني:
**مين فيه يشوف المشروع (repository) تبعك**

---

# 🔓 Public (عام)

إذا المشروع **public**:

👉 أي حدا بالعالم فيو:

* يشوف الكود 👀
* ينسخو (clone) 📥
* يستفيد منه 💡

## 🎯 متى تستعمله؟

* مشاريع للتعلّم
* portfolio تبعك
* open source

💡 مثلاً: عم تتعلّم Next.js، فيك تخلي المشروع public لحتى تفرجي شغلك

---

# 🔒 Private (خاص)

إذا المشروع **private**:

👉 بس انت (وأشخاص بتضيفهم) فيهم:

* يشوفوا الكود
* يشتغلوا عليه

## 🎯 متى تستعمله؟

* مشروع شغل 🏢
* كود سرّي 🤫
* شي ما بدك حدا يشوفه

---

# 🤯 الفرق السريع

| النوع   | مين بيشوفه       |
| ------- | ---------------- |
| Public  | الكل 🌍          |
| Private | بس انت والتيم 👥 |

---

# 💡 مثال عليك

* عم تتعلّم → خليو **public** ✅
* عم تشتغل مع شركة → خليو **private** ✅

---

# 🧠 الخلاصة

👉 **visibility = مين مسموح يشوف المشروع**
