## أول شي: شو هو Git؟

**Git** هو نظام لإدارة الإصدارات (Version Control System)

يعني ببساطة:

* بيخلّيك تحفظ كل التعديلات يلي بتعملها على الكود
* فيك ترجع لأي نسخة قديمة
* فيك تشتغل مع فريق بدون ما تخربوا على بعض

💡 فكّر فيه متل “history” للكود تبعك

### مثال:

* كتبت كود
* عملت commit → حفظت نسخة
* خرب الكود → فيك ترجع للنسخة القديمة بسهولة

---

## تاني شي: شو هو GitHub؟

**GitHub** هو موقع بيستخدم Git

يعني:

* Git = الأداة (local على جهازك)
* GitHub = المكان يلي بترفع عليه الكود (online)

### شو بيقدملك GitHub؟

* تخزين المشاريع (repositories)
* مشاركة الكود مع غيرك
* العمل الجماعي (collaboration)
* Pull Requests (للمراجعة قبل الدمج)

---

## الفرق بسرعة ⚡

* Git → شغل على جهازك
* GitHub → شغل أونلاين + مشاركة

---
