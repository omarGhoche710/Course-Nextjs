تمام 🔥 هيدي من أقوى الأدوات لتفهم شو تغيّر بالكود

---

# 🧠 شو يعني `git diff`؟

👉 **`git diff` = يفرجيك الفرق بين نسختين من الكود**

---

# 🎯 أبسط استخدام

```bash
git diff
```

👉 بيورجيك:

* التعديلات يلي عملتها
* بس **لسا ما عملت commit إلها**

---

# 🧪 مثال

قبل:

```js
let x = 5;
```

بعد:

```js
let x = 10;
```

---

## `git diff` بيطلع:

```diff
- let x = 5;
+ let x = 10;
```

---

# 🤯 شو يعني الرموز؟

* `-` = السطر القديم
* `+` = السطر الجديد

---

# 🔥 استخدامات مهمة

## 1️⃣ تشوف تغييراتك قبل commit

```bash
git diff
```

---

## 2️⃣ تشوف شو ضفت على staging

```bash
git diff --staged
```

👉 الفرق بين:

* `add`
* و `commit`

---

## 3️⃣ مقارنة بين branchين

```bash
git diff main feature-login
```

👉 شو الفرق بيناتهم

---

## 4️⃣ بعد `fetch`

```bash
git diff main origin/main
```

👉 شو التيم عدّل على GitHub

---

# 🎮 مثال بالحياة

👤 انت عمs تشتغل
قبل ما تعمل commit:

```bash
git diff
``` 

👀 بتراجع شغلك
بعدين:

```bash
git add .
git commit -m "clean code"
```
