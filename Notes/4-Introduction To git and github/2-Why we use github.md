سؤال مهم 🔥 لأن إذا فهمت شو بيفيدك **GitHub** صح، بتستفيد منه كتير كمبرمج.

خلّيني أبسّطلك 👇

---

## 🔹 1. حفظ مشاريعك أونلاين

بدل ما الكود يكون بس على اللابتوب:

* إذا خرب جهازك → ضاع كل شي 😅
* مع GitHub → كل شي محفوظ أونلاين

---

## 🔹 2. تشتغل مع غيرك (Team Work)

إذا عم تشتغل مع رفقاتك:

* كل واحد بيشتغل على جزء
* بعدين بتدمجوا الشغل

💡 هون بيجي دور:

* commits
* branches
* pull requests

---

## 🔹 3. Portfolio لإلك 💼

GitHub متل CV للمبرمج

لما تقدّم على شغل:

* بيشوفوا مشاريعك
* بيشوفوا كيف بتكتب كود
* بيعرفوا مستواك

---

## 🔹 4. ترجع لنسخ قديمة (Version Control)

إذا خرب الكود:

* فيك ترجع لأي commit قديم بسهولة

(هاي ميزة جاية من **Git**)

---

## 🔹 5. تعلّم من غيرك

في ملايين المشاريع:

* فيك تشوف كيف ناس أقوياء بيكتبوا كود
* تنسخ أفكار (مش copy paste 😄، تفهم وتتعلم)

---

## 🔹 6. تنزل مشاريع جاهزة (clone)

مثلاً:

* لقيت project حلو
* بتعمل clone وبتشتغل عليه

---

## 🔹 7. تنشر مشاريعك (Deploy)

فيك تستخدم:

* GitHub Pages

لتنشر:

* مواقع static (HTML / CSS / JS)

---

## 🔹 8. توثيق وتنظيم الشغل

فيك تضيف:

* README (شرح المشروع)
* Issues (مشاكل)
* Projects (تنظيم المهام)

---

## 🔥 الخلاصة

GitHub بيساعدك:

* تخزّن مشاريعك
* تشتغل مع فريق
* تبني portfolio
* تتعلم من غيرك
* ترجع لأي نسخة
* وتنشر مشاريعك

---
