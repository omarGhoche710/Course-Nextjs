تمام 🔥 خلّيني أبسّطلك كلمة **Repositories** بطريقة سهلة:

## شو يعني Repository؟

**Repository** (أو repo اختصارًا) هو:

👉 **مكان بتخزّن فيه مشروعك كامل**

---

## يعني شو “مشروعك كامل”؟

الـ repository بيحتوي:

* كل ملفات الكود (HTML, CSS, JS…)
* الصور والملفات
* تاريخ التعديلات (commits)
* معلومات المشروع

💡 يعني متل **folder كبير ذكي** بس مش عادي، بيعرف كل التغييرات يلي صارت

---

## مثال بسيط 👇

إذا عندك موقع:

```
my-website/
  index.html
  style.css
  script.js
```

هيدا كله بيتخزن داخل **repository واحد**

---

## وين بيكون؟

في نوعين:

### 1. Local Repo

على جهازك باستخدام **Git**

### 2. Remote Repo

أونلاين على **GitHub**

---

## شو بتعمل فيه؟

داخل الـ repository فيك:

* تعمل commit (تحفظ نسخة)
* تعمل push (ترفع ع GitHub)
* تعمل pull (تنزل تحديثات)
* تعمل branch (تجرب بدون ما تخرب الأساسي)

---

## مثال من الحياة 🧠

تخيّل:

* Repository = ملف مشروعك بالجامعة
* كل commit = نسخة محفوظة من شغلك
* GitHub = Google Drive تبع هيدا المشروع
