## Naming Components

- "/" => Home
- "/about" => AboutPage
- "/products/[productId]" => ProductDetails ([productId] more descriptive than [id])
- "/products/[productId]/reviews/[reviewsId]" => ProductReview
- "/docs/[[...slug]]" => Docs
- not-found.tsx => NotFound
- "/blog" => Blog
- "(auth)/layout.tsx" => AuthLayout
- "articles/[articleId]" => NewsArticle
- "error.tsx" => ErrorBoundary
- "error-wrapper.tsx"=> ErrorWrapper
- "complex-dashboard" => ComplexDashboardPage
